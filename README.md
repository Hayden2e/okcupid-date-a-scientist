This is a ML portfolio project using the OKCupid Profiles data available on Kaggle. This data is not included within the repository due to file size limits.

In this project, I aim to predict users' orientation based on their other profile data. I first carry out some exploratory data analysis, then construct a logistic regression model and optimise it by developing prediction thresholds, trialling different regularization methods and optimizing the regularization strength, and feature engineering. This produced little gains so I also carry out natural language processing on the essay data using TF-IDF to improve the results.

This project will be concluded with a neural network to (hopefully!) produce more accurate predictions.
